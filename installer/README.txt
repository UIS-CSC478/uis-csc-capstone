System Requirements
	Operating System: Windows 10 or Windows 11
	Memory: Minimum 4 GB RAM
	Disk Space: At least 500 MB free space
	Additional Software: None required
	(Java is bundled with the application)

Scrabble App Installation
	1. Extract the zip file and open it.
	2. Double-click install-scrabble.bat.
	3. The installer will copy the application to your detected Desktop and launch it.
	4. After installation, you can run the application anytime by opening:
		Desktop > Scrabble App > Scrabble App.exe

	No additional software is required.

Known Issues
	The installer is designed for Windows systems only and will not run on macOS or Linux.
	Some systems may display a Windows SmartScreen warning when running the .bat file. 
		If this occurs, click 'More Info' > 'Run Anyway'.

Basic Troubleshooting
	Installer does not Run:
		Right-click the file and select Run as administrator if necessary
	Application does not launch after install:
		Navigate to: Desktop > Scrabble App
		Double-click: Scrabble App.exe
