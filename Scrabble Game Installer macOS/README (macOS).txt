** SCRABBLE GAME APP - INSTALLATION GUIDE (macOS) **



System Requirements
	
	
	* Operating System: macOS 11 or newer
	* Memory: Minimum 4 GB RAM
	* Disk Space: At least 500 MB of free space
	* Additional Software: None required
	(Java is bundled with the application)


Installation

	* Extract the zip file and open it.
	* Double-click to open "Scrabble.dmg."
		- A window will open, showing "Scrabble App"

	* Drag "Scrabble App" into the Applications folder.
	
	* Open the Applications folder.

	* Double-click "Scrabble App" to launch.


** macOS may block the application on its first launch because it is not from an identified developer. **

	If this happens,
		
		* Right-click the app and select "Open"
		* Click "Open" again


Troubleshooting:

	
	If the application crashes upon launch, reinstall the app from the .dmg file, ensuring that "Scrabble App" is dragged into the Applications folder.
